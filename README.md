# SuperBonus110
Dati rilasciati da Enea relativi all'andamento del Superbonus 110%

A partire dall'agosto 2021, Enea pubblica report mensili relativi ai costi legati al Superbonus 110%

https://www.efficienzaenergetica.enea.it/detrazioni-fiscali/superbonus/risultati-superbonus.html

A partire da questi dati, ho generato due file:
Andamento nazionale mostra i valori mensili relativi al costo su base nazionale
Valori regionali al 31_3_24 mostra invece il costo su base regionale a fine marzo 2024, con un calcolo del costo pro capite
